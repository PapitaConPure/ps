Descripción
Ctrl + /
Muestra la ventana de atajos de teclado.
Editor
Ctrl + Enter
Ejecuta el código PuréScript ingresado en el editor.
Ctrl + Shift + E
Exporta el código PuréScript ingresado en el editor.
Ctrl + Shift + H
Abre la página principal de la documentación oficial de PuréScript.
F1
Abre la página principal de la documentación oficial de PuréScript (pueden haber problemas con este atajo).
Caja de Entradas
Enter
Re-ejecuta el programa escrito en el editor con las entradas indicadas.
Caja de Entradas
Shift + Enter
Baja un renglón en la caja de entradas sin re-ejecutar el programa.
Editor
Ctrl + Espacio
Muestra sugerencias de autocompletado en base a lo que se esté escribiendo actualmente.
Editor
Tab
Si hay líneas seleccionadas, aumenta la sangría de esas líneas. De lo contrario, agrega una sangría en la posición actual del cursor.
Editor
Shift + Tab
Si hay líneas seleccionadas, reduce la sangría de esas líneas. De lo contrario, reduce la sangría de la línea actual.
Editor
Ctrl + ← , Ctrl + →
Desplaza el cursor entre palabras y símbolos.
Editor
Alt + ← , Alt + →
Desplaza el cursor al inicio y fin de la línea actual respectivamente.
Editor
Ctrl + C
Si hay texto seleccionado, copia ese texto al portapapeles. De lo contrario, copia toda la línea actual al portapapeles.
Editor
Ctrl + V
Si el portapapeles contiene una línea de código, se pega esa línea por encima de la línea actual. De lo contrario, se intenta pegar texto del portapapeles en la posición actual del cursor.
Editor
Ctrl + X
Si hay texto seleccionado, corta ese texto al portapapeles. De lo contrario, corta toda la línea actual al portapapeles.
Editor
Ctrl + A
Selecciona todo el texto ingresado en el editor.
Editor
Ctrl + D , Shift + Supr
Elimina la línea actual junto con sus contenidos.
Editor
Shift + Supr
Elimina la palabra o símbolo que le sigue al cursor.
